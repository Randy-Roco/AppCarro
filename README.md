# AppCarro — Calibración por Colimación y Cálculo de Puntos

Aplicación web local para **geomensura/topografía**:
- Calibra offsets **ΔAH (azimut)** y **ΔAV (inclinación)** usando puntos de colimación con coordenadas conocidas.
- Calcula coordenadas (X,Y,Z) de nuevos puntos ingresando AH/AV/Distancia.
- Exporta resultados a **TXT** (CSV/TSV), ideal para Excel y/o carga GIS.
